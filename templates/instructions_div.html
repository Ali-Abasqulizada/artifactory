Artifact database is empty

You can add artifacts by pushing them using this curl command:

curl -F "name=example_name" \
     -F "commit_hash=example_hash" \
     -F "file=@/path/to/your/file" \
     -F "tags={\"deployed\": true}" \
     http://localhost:5000/push

You can downloads artifacts using this curl command:

curl "http://127.0.0.1:5000/download?name=example_name" -o new_file

Using both name and commit_hash

curl "http://127.0.0.1:5000/download?name=example_name&commit_hash=example_hash" -o new_file1

You can see the differences between artifacts using this curl command:

curl "http://127.0.0.1:5000/diff?name=example_name&commit_hash_a=example_hash_a&commit_hash_b=example_hash_b"
